# Project-X

Backend ve frontend bileşenlerine sahip çok platformlu bir uygulama projesi.

## ⚠️ UYARI: Geliştirme Kuralları

**Lütfen aşağıdaki kurallara mutlaka uyun:**
- ✅ Commit mesajlarını anlamlı yazın
- ✅ Push işlemlerini günün sonunda yapın
- ✅ Push yapmadan önce kodunuzu test edin
- ✅ Gereksiz dosyaları commit etmeyin
- ❌ Çalışmayan kod push etmeyin
- ❌ Gün ortasında push yapmaktan kaçının


### Commit Mesajları
Commit mesajlarınızı anlamlı ve açıklayıcı yazın:
- `feat: yeni özellik eklendi`
- `fix: hata düzeltildi`
- `docs: dokümantasyon güncellendi`
- `style: kod formatı düzenlendi`
- `refactor: kod yeniden yapılandırıldı`

### Push Öncesi Kontrol Listesi
Push yapmadan önce:
- [ ] Kodunuzun hatasız çalıştığından emin olun
- [ ] Test dosyalarını kontrol edin
- [ ] Gereksiz dosyaların commit edilmediğinden emin olun
- [ ] Commit mesajınızın açıklayıcı olduğundan emin olun
- [ ] Değişikliklerinizi gözden geçirin
